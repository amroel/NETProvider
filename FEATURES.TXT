FB.NET - Firebird managed data provider for .NET and Mono
=========================================================

List of features.

- Written 100% in C#, using managed code.

- Support for Firebird 1.0, Firebird 1.5 and Interbase 6.0 servers.

- Firebird Services:

	* Backup.

	* Configuration.

	* Log.

	* Restore.

	* Security.

	* Server properties.

	* Statistical.

	* Validation.

- Basic ISQL scripts execution.

- Data types:

	* Array.
	* BigInt.
	* Binary and Text Blobs.
	* Char and Varchar.
	* Date.
	* Integer.
	* SmallInt.
	* Time.
	* TimeStamp.

- Connection Pooling.

- Command builder.

- Named parameters, an example:

	FbConnection connection = new FbConnection(connectionString);
	FbCommand command = new FbCommand("select * from table_name were field_name = @param_value");
	FbDataReader reader = command.ExecuteReader();
	while(reader.Read)
	{
		...
	}
	reader.Close();
	connection.Close();

- Batch command execution, an example:

	FbConnection connection = new FbConnection(connectionString);
	FbCommand command = new FbCommand("select * from table_name1;select * from table_name2");
	FbDataReader reader = command.ExecuteReader();
	while (reader.NextResult())
	{
		...
		while (reader.Read)
		{
			...
		}
	}
	reader.Close();
	connection.Close();

- Stored procedure execution using SQL Server-like syntax, an example:

	FbConnection connection = new FbConnection(connectionString);
	FbCommand command = new FbCommand("sp_select_data");
	command.CommandType = CommandType.StoredProcedure;
	FbDataReader reader = command.ExecuteReader();
	while(reader.Read)
	{
		...
	}
	reader.Close();
	connection.Close();
